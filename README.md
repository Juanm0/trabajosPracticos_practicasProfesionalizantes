Consignas de evaluación
Utilizando los conocimientos que adquiridos por el estudiante en HTML y CSS deberán Realizar
un archivo horarios.html, el cual deberá contener el desarrollo indicado en las siguientes
consignas:
1. Realizar el horario de la carrera de sistemas.
2. Debe tener el mismo formato de texto que el horario real:
a. Materia en negrita
b. Profesor en cursiva
c. Encabezados con diferente color a los datos
d. Espacios sin clases deben tener su propio color diferenciador
3. Los estilos deben estar realizados en el archivo style.css.
4. Cuando el mouse pase por una casilla, esta debe tener un un color suave.
